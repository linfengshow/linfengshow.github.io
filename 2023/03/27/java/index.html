（补充）JAVA的内存结构

Java内存结构分析：
  • 1.栈：存放基本数据类型（局部变量）
  • 2.堆：存放对象
  • 3.方法区：常量池（常量，比如字符串），类加载信息
Java创建对象流程的简单分析：
  • 1.先加载类信息（属性，方法）==一个类只加载一次==
  • 2.在堆中分配空间，并默认初始化
  • 3.把空间的地址分配给p1,p1指向该地址
  • 4.进行指定初始化
